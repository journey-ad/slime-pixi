# 原神史莱姆乐园 - PIXI.JS实现

使用 PIXI.JS 复刻的的原神史莱姆乐园活动~~私服~~

## 技术栈

- Vue.js
- Pixi.js
- Pixi Spine
- GSAP(TweenMax)
